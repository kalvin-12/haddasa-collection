# Haddasa Collection

Website profil dan katalog produk **Haddasa Collection**, sebuah brand modest fashion yang mengedepankan keanggunan dan kualitas premium. Website ini dirancang dengan estetika modern menggunakan referensi template kreatif **Tank**.

## ✨ Fitur Utama
- **Desain Modern & Elegan**: Menggunakan Bootstrap 5 untuk tata letak yang bersih dan responsif di semua perangkat.
- **Animasi GSAP**: Efek transisi yang halus, *reveal text*, dan interaksi cursor yang interaktif.
- **Navigasi Cerdas**: 
  - Navbar yang bersembunyi saat scroll ke bawah dan muncul kembali saat scroll ke atas.
  - Menu pop-up *full-screen* yang minimalis.
- **Katalog Produk**: Tampilan produk dengan rasio portrait (2:3) yang konsisten dan rapi.
- **Halaman Terpisah**: Struktur halaman mandiri untuk Home, Shop, Shop Details, Collections, About, dan Contact.

## 🛠️ Teknologi yang Digunakan
- **HTML5 & CSS3**
- **Bootstrap 5** (Layout & Responsivitas)
- **GSAP (GreenSock Animation Platform)** (Animasi)
- **Font Awesome 6** (Ikon)
- **Google Fonts** (Playfair Display & Poppins)

## 📁 Struktur Proyek
```text
haddasa/
├── assets/
│   ├── css/
│   │   └── style.css          # Kustomisasi gaya visual utama
│   ├── js/
│   │   └── script.js         # Logika animasi GSAP dan interaksi UI
│   └── img/                  # Direktori penyimpanan gambar/aset visual
├── index.html                # Halaman Beranda (Home)
├── shop.html                 # Halaman Katalog Produk (Shop)
├── shop-details.html         # Halaman Detail Produk
├── collections.html          # Halaman Koleksi Seri
├── about.html                # Halaman Tentang Kami (Our Story)
├── contact.html              # Halaman Kontak
├── vercel.json               # Konfigurasi deployment Vercel (Clean URLs, dll)
└── README.md                 # Dokumentasi proyek ini
```

## 🚀 Deployment
Website ini siap di-deploy ke **Vercel**. Dengan file `vercel.json` yang tersedia, URL akan secara otomatis menjadi bersih tanpa ekstensi `.html` (contoh: `/about` alih-alih `/about.html`).
